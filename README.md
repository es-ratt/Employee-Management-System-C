# Employee-Management-System-C
**Employee Management System in C**

This project is a console-based Employee Management System developed using the C programming language as part of a capstone course. It demonstrates the implementation of a real-world management system using structured programming concepts in C.
<br>
The system supports two user roles, Admin and HR, with role-based access control. It includes user authentication and manages employee data using file handling for persistent storage.
<br>
Key functionalities include adding new employee records, viewing employee details, updating employee information, managing attendance, and calculating salaries with bonus and deduction logic based on attendance.
<br>
This project applies core C programming concepts such as structures, functions, file handling, string manipulation, and modular program design to simulate a basic organizational employee management system.
